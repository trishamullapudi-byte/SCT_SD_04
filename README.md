# Web Scraper Project
This project is a Python-based web scraper that extracts book details from a website and stores
the data in a structured format. The scraper collects information such as book title, price,and
rating and saves it into a CSV file for further analysis.

# Features
Extracts book information from a website

Collects title, price, and rating

Stores data in CSV format

Simple and efficient data extraction using Python
